Predix Time Series Datasource Configuration

  • Predix TS URL typically this is https://time-series-store-predix.run.aws-usw02-pr.ice.predix.io
  • Predix Zone ID The Predix Zone Id of your instance (run: cf env MyApp to get this value)
  • UAA URL The UAA URL of your Predix UAA provider
  • Client id and Secret The clientID and client secret in the format clientID:clientSecret for the UAA provider
  • Response Type The UAA Response type - usually: token
  • Grant Type The UAA Grant type - usually: client_credentials
  • Origin This will be put into the Access-Control-Allow-Origin header for CORS compliance
